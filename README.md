\# Health Status Prediction using Machine Learning



This project applies supervised machine learning techniques to predict health status based on lifestyle and medical features.



\## Project Overview



The workflow includes:



\- Data preprocessing

\- Exploratory Data Analysis (EDA)

\- Feature scaling

\- Training multiple ML models

\- Hyperparameter tuning

\- Ensemble methods

\- Model evaluation using confusion matrix and ROC curve



\## Models Implemented



\- Logistic Regression

\- Decision Tree Classifier

\- Bagging Classifier

\- AdaBoost Classifier

\- Gradient Boosting Classifier

\- Support Vector Classifier (Best Model)



\## Best Model Performance



Support Vector Classifier achieved the best performance:



Test Accuracy: \*\*94.66%\*\*
